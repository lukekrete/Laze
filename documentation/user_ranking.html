User Ranking Requirements and Privileges

Rank Requirements Privileges
Moderator
  • Profile must be manually reviewed first
  • +1000 or higher points
  • Ability to view pins
  • Create unlimited number of pins per day
  • Created pins take priority when pins are created from lower ranks at the same location
  • Upvote or downvote unlimited pins per day
  • View and assess reported pins
  • Delete pins if necessary
Regular
  • +100 points
  • Ability to view pins
  • Create 25 pins per day
  • Created pins take priority when pins are created from lower ranks at the same location
  • Upvote or downvote 100 pins per day
  • Can report pins
Neutral
  • Default rank given at time of account creation
  • Ability to view pins
  • Create 5 pins per day
  • Created pins take priority when pins are created from lower ranks at the same location
  • Upvote or downvote 20 pins a day
  • Can report pins
Probation
  • -100 points
  • Ability to view pins
  • Create 1 pin per day
  • Cannot report pins
Muted
  • -300 points
  • Ability to view pins
  • Cannot create any new pins
  • Cannot report pins

**Each pin begins with a score of 1 point, upvotes from other users add 1 point, downvotes from other users subtracts 1 point, votes from a single person on multiple pins created by the same person can only add/subtract to/from a person’s points at a maximum of 2 a month, this is to deter one person being able to ruin the rank or boost a rank of another

*If a pin loses priority because of someone with a higher rank creating the same pin with the same info, if the pin earns positive points it goes to the lower rank user, if the pin earns negative points it goes to the higher rank user, this will deter higher ranked users from “stealing” points from lower ranked users
